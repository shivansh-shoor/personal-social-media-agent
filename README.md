# Personal Social Memory Agent

**OpenEnv Round 1 Submission** — Shivansh Dev Shoor

An environment where an AI agent learns to build personal social memory from Instagram-style posts, Threads discussions, and WhatsApp chat logs.

### Tasks
- **Fact Extraction** (Easy): Extract specific facts from posts/chats
- **Thread Summary** (Medium): Summarize conversations
- **Multi-hop QA** (Hard): Connect information across multiple sources

### How to Run
```bash
HF_TOKEN=your_token HF_SPACE_URL=https://your-space.hf.space python inference.py
```
Built for Meta PyTorch × Hugging Face OpenEnv Hackathon.
